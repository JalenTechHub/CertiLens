CertiLens Pro
A multi-layered real-time phishing detection engine built as a Chrome Manifest V3 extension. Combines Certificate Transparency monitoring, DNS security auditing, DOM heuristics, and threat intelligence to replicate the core logic of enterprise EDR browser agents.
Chrome MV3 Vanilla JS Certificate Transparency RDAP / WHOIS DNS over HTTPS URLScan.io Homograph Detection DOM Heuristics
Architecture
⚙️
background.js
Service worker orchestrating all API calls in parallel. Implements caching, retry logic, and risk score computation.
🖥️
content.js
Injected into every page. Scans DOM for password fields on HTTP, hidden iframes, obfuscated scripts, and phishing form actions.
📊
popup.js
Lightweight UI controller. Renders animated risk ring, engine cards, and findings list from background analysis results.
🔏
CT Logs (crt.sh)
Queries Certificate Transparency logs. Certs issued in <30 days are a strong phishing signal — attackers can't buy old certs.
🌐
RDAP + DoH
Domain age via RDAP (replacement for WHOIS). DNS over HTTPS (Cloudflare) checks SPF/DMARC email authentication records.
🔤
Homograph Engine
Detects IDN homograph attacks: Cyrillic 'а' vs Latin 'a', digit substitution ('0' → 'o'), and mixed Unicode script domains.
